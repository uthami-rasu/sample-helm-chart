# Helm - Full-Stack Task Manager (SoftQA)

A modern, premium full-stack task management application built with **Flutter Web** and **Java Spring Boot**.

## 🚀 Project Overview

Helm is designed with the "SoftQA" aesthetic, focusing on a clean, professional, and responsive user experience. It features a complete authentication system and full CRUD capabilities for task management.

### 📁 Project Structure

- **`frontend/`**: The Flutter Web application.
- **`backend/`**: The Java Spring Boot 3 server with JWT security and H2 database.
- **`docs/`**: Detailed architecture and configuration guides.

---

## 🛠️ Quick Start

### 1. Start the Java Backend

```bash
cd backend
mvn spring-boot:run
```

_The backend will be available at `http://localhost:8080`_

### 2. Start the Flutter Frontend

```bash
cd frontend
flutter pub get
flutter run -d chrome
```

---

## ✨ Features

- **Premium UI**: Modern dark teal aesthetic with smooth animations and responsive layouts.
- **Secure Auth**: JWT-based login and registration system.
- **Task Management**: Create, view, edit, delete, and toggle tasks with real-time stats.
- **Reliable Backend**: Spring Security protected API with JPA persistence.
- **SoftQA Branding**: Professionally branded interface with modern typography.
